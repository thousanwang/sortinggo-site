v1.5.6 2026-07

  • La entrada de pedidos por voz ahora admite el campo de especificación en los planes Std / Pro, además del nombre del producto, la cantidad y la unidad
  • Se agregó análisis de especificaciones para todos los idiomas de reconocimiento de voz admitidos: chino tradicional, chino simplificado, japonés, inglés, coreano, vietnamita, tailandés, indonesio y español
  • Se agregó ayuda en el Step 2 para mostrar los campos disponibles y ejemplos de cómo hablar según el idioma de reconocimiento actual y el plan Free / Lite / Std / Pro
  • La configuración de unidades se amplió a unidades / especificaciones, para que el usuario pueda administrar las palabras de especificación reconocidas por la entrada de voz
  • Se agregó confirmación de ambigüedad de especificación: cuando una frase puede ser un nombre completo de producto o producto + especificación, el usuario puede conservar el nombre o separarlo como especificación
