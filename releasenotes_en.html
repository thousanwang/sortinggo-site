v1.5.9 2026-07

  • Changed version checks to read SortingGo's website version.json and guide users to Google Play when an update is available
  • Improved startup reliability when Google Play or the website is temporarily unreachable, so version checks no longer block sorting work
  • Added a short fallback message when release notes cannot be loaded: "A new version is available. Please update from Google Play."
  • Fixed an issue where hands-free voice sorting could stop working after switching SortingGo to the background or another app because the microphone could not start listening
  • Fixed cases where Android recent apps or foreground service notifications could show the default Android icon

v1.5.8 2026-07

  • Pro plan voice-controlled hands-free sorting is now available for purchase, no longer marked "Coming soon"
  • Added an auto-listen attempts setting (1-5, default 2): after each voice prompt, listening automatically retries this many times when no command is recognized
  • Added Bluetooth headset audio routing for voice commands, improving mic pickup and playback when using a paired headset
  • Added a stop-listening control to the voice status bar
  • Voice sorting now keeps running when the screen turns off, and only stops when you leave the app, enabling truly hands-free operation
  • Added a "Feature unavailable" dialog that guides Free/Lite/Std users to the Subscription page when they try to use voice sorting

v1.5.7 2026-07

  • Improved startup reliability when Google Play Store or Play services are temporarily unavailable: in-app update checks now fail silently and never block offline sorting
  • Improved subscription restore safety: failed Google Play Billing queries no longer overwrite a paid local plan as Free
  • Added a 45-day subscription verification grace period: paid users can keep working offline and will only see a dismissible reminder to reconnect and verify

v1.5.6 2026-07

  • Voice order input now supports the spec field on Std / Pro plans, in addition to product name, quantity, and unit
  • Added multilingual spec parsing for all supported voice languages: Traditional Chinese, Simplified Chinese, Japanese, English, Korean, Vietnamese, Thai, Indonesian, and Spanish
  • Added Step 2 voice order help showing Free / Lite / Std / Pro supported fields and how to speak units and specs in the selected recognition language
  • Added editable voice spec options alongside voice unit options, so users can customize spec words used by voice input
  • Added an ambiguity confirmation flow: when a phrase may be either a full product name or product + spec, users can keep the product name or split it into a spec

v1.5.5 2026-07

  • Fixed update-ready banner position: now correctly appears at the bottom of the screen instead of the top
  • Added a "Later" dismiss button to the update-ready banner so users can choose when to restart and apply the update
  • Fixed duplicate product name in the Simplified Chinese voice order demo hint text
  • Voice order input now supports two word orders: A. name + quantity + unit (e.g. "cabbage 40 boxes"); B. quantity + unit + name (e.g. "40 boxes cabbage")
