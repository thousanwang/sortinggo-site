v1.5.6 2026-07

  • Voice order input now supports the spec field on Std / Pro plans, in addition to product name, quantity, and unit
  • Added multilingual spec parsing for all supported voice languages: Traditional Chinese, Simplified Chinese, Japanese, English, Korean, Vietnamese, Thai, Indonesian, and Spanish
  • Added Step 2 voice order help showing Free / Lite / Std / Pro supported fields and how to speak units and specs in the selected recognition language
  • Added editable voice spec options alongside voice unit options, so users can customize spec words used by voice input
  • Added an ambiguity confirmation flow: when a phrase may be either a full product name or product + spec, users can keep the product name or split it into a spec

v1.5.5 2026-07

  • Fixed update-ready banner position: now correctly appears at the bottom of the screen instead of the top
  • Added a "Later" dismiss button to the update-ready banner so users can choose when to restart and apply the update
  • Fixed duplicate product name in the Simplified Chinese voice order demo hint text
  • Voice order input now supports two word orders: A. name + quantity + unit (e.g. "cabbage 40 boxes"); B. quantity + unit + name (e.g. "40 boxes cabbage")
