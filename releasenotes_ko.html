릴리스 노트

v1.5.4 2026-06

  • 시작 시 Google Play Billing 연결 실패로 '이 플랜 구독하기' 버튼이 영구적으로 비활성화되는 문제 수정; 연결 오류 메시지 및 다시 시도 버튼 추가
  • 업데이트 알림 개선: 새 버전이 감지되면 다운로드를 권유하기 전에 앱 내에서 릴리스 노트를 표시

v1.5.3 2026-06

  • Google Play 유연한 인앱 업데이트 추가: 새 버전을 백그라운드에서 다운로드하고 완료되면 화면 하단에 재시작 안내 표시
  • 도움말 허브에 '릴리스 노트' 항목 추가하여 버전 기록 확인 가능
  • 도움말 허브에 '버그 신고' 기능 추가 (언어·구독·버전 정보가 이메일에 자동 입력)
  • 도움말 허브에 '기능 요청' 기능 추가 (사용 정보가 이메일에 자동 입력)
  • 도움말 허브에 'Google Play 평가하기' 항목 추가
  • 지원 웹사이트에 Facebook 공식 페이지 연락처 추가

v1.5.2 2026-06

  • 홈 화면에서 언어 전환이 간헐적으로 실패하는 문제 수정 (Android 13 / API 33 이상)
  • 튜토리얼 1단계의 공식 웹사이트 링크 복원 (영어, 스페인어, 태국어, 베트남어, 인도네시아어)
  • 스페인어 화면의 텍스트 깨짐 및 홈 메뉴 번호 오류 수정
